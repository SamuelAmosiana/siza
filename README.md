# SIZA – Study In Zambia

**SIZA** is a centralised higher-education discovery platform designed to bridge the information gap between prospective students and Zambian universities. Students can browse institutions, compare academic programmes, and submit initial interest applications — all from one unified, enterprise-grade platform.

---

## 🌟 Key Features

- **University Directory:** Comprehensive listings of verified Zambian higher-education institutions (Public & Private).
- **Advanced Programme Search:** Discover academic programmes filtered by level (Bachelor, Master, Diploma, etc.), faculty, or institution.
- **Smart Application System:** Submit "Initial Interest" applications directly to admissions offices with unique tracking references.
- **Enterprise Aesthetics:** A premium, responsive UI built with an Emerald-themed design system, optimized for both desktop and mobile users.
- **Inertia Powered:** Seamless, single-page application feel without the complexity of a separate API layer.

---

## 🛠️ Technology Stack

| Layer | Technology |
|---|---|
| **Backend** | PHP 8.2+ / Laravel 12 |
| **Frontend** | Vue 3 (Composition API) |
| **Bridge** | Inertia.js |
| **Styling** | Tailwind CSS v3 |
| **Database** | MySQL / Eloquent ORM |
| **Build Tool** | Vite |

## 🚀 Getting Started

### Prerequisites
- PHP 8.2 or higher
- Composer
- Node.js & NPM
- MySQL (via XAMPP or similar)

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/SamuelAmosiana/siza.git
   cd siza
   ```

2. **Install dependencies:**
   ```bash
   composer install
   npm install
   ```
